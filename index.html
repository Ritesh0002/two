Recent Projects

Project 1

Furniture Website

The website includes a home page, product listing pages, product detail pages, and a contact page. For the website design, I used a modern and clean aesthetic with a color scheme that reflects the company's brand. Project 2

Gym website

My gym website project is a web application designed to provide information about a gym and its services to potential customers. The website includes several pages, such as a homepage, about us page, services page, class schedule page, and a contact page. The services page provides an overview of the gym's offerings, including personal training, group classes, and nutrition coaching. The class schedule page displays the gym's current class schedule and allows users to filter classes by type and time.!

Project 3

Solar Energy

The solar energy website project is a web development project I worked on for a client in the renewable energy industry. The project involved designing and developing a website that promotes the client's solar energy services and educates visitors about the benefits of solar energy. To achieve this goal, I used a variety of technologies and techniques, including HTML, CSS, JavaScript, and Bootstrap. I created a responsive design that is optimized for both desktop and mobile devices, and incorporated interactive elements such as animations and sliders to engage visitors.!

Project 4

Tours and Travel

The Tours and Travel website project is a responsive and visually appealing website designed for a travel agency. The website provides information about various destinations, tour packages, and travel services offered by the agency. The website includes features such as a homepage with a slider showcasing popular destinations, a search bar for finding tours and packages, a gallery section with images and descriptions of destinations, a contact form for inquiries and bookings, and a blog section with articles and tips for travelers.!
